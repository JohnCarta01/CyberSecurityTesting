# Cyber Security Testing - University of Oulu, 2025

This repository contains my assignments and notes for the Cyber Security Testing course at the University of Oulu in 2025.

## Structure

- [Week1](Week1/) - Introduction to cyber security testing and the importance of thinking models, expertise, and automation in security testing.
- [Week2](Week2/) - Data input security, with exercises on a personal essay, a ChatGPT revision, and a follow-up report.
- [Week3](Week3/) - Web security testing, with a report collected in the week folder for the course exercise.
- [Week4](Week4/) - Practical lab work on security testing exercises, with reports and notes from the work completed during the laboratory sessions.
- [Week5](Week5/) - Data input security, focusing on command injection, input validation, and a file upload / Zip Slip style challenge.
- [Week6](Week6/) - Fuzzing and vulnerability discovery, including an AFL++ buffer-overflow analysis and a ffuf-based login exercise.
- [Week7](Week7/) - Final-week assignment template, prepared for the remaining course tasks.
- [img](img/) - Shared images used across the repository.

Each week folder contains the related write-up and, where relevant, the supporting screenshots, PDFs, or other attachments used in the report.
